

# Task Manager Application

This is a simple Task Manager application built using **Spring Boot** and **PostgreSQL**.
It allows users to manage tasks, categories, priorities, and users with proper backend structure and database integration.

The project focuses on clean architecture, REST APIs, and proper exception handling.

---

## Features

* Create, update, delete, and view tasks
* Manage categories and priorities
* Assign tasks to users
* Global exception handling
* Layered architecture (Controller → Service → Repository)
* Database integration using JPA & Hibernate

---

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Maven
* Lombok
